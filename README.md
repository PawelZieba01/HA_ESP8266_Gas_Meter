# HA_ESP8266_Gas_Meter

Projekt prostego loggera zbierającego dane na temat zużycia gazu w gospodarstwie domowym.

Dane raportowane są do serwera smart-home opartego o syatem HomeAssistant.

Prezentacja danych za pomocą aplikacji Grafana.

Komunikacja z serwerem za pomocą protokołu MQTT

Język: MicroPython
